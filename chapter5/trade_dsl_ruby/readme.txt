Running the examples:
--------------------

Prerequisite: Download and install JRuby (http://jruby.org). The current example is built using JRuby 1.5.1.

From each of the folders lib/iteration1, lib/iteration2 and lib/iteration3 do the following:-

a. mvn clean
b. mvn install ## will generate class files in current directory
c. java -cp d:\tech\jruby-1.5.1\lib\jruby.jar;. run_dsl 

Note : Replace d:\tech\jruby-1.5.1\lib\jruby.jar with your installation path of JRuby

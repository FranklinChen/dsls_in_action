Running the example:
--------------------

From folder "trading", do the following:-


a. mvn clean
b. mvn compile
c. mvn exec:java -Dexec.mainClass="ch8.order.parsing.dsl.Main"
d. mvn exec:java -Dexec.mainClass="ch8.order.semantic.dsl.Main"
e. mvn exec:java -Dexec.mainClass="ch8.order.ssi.dsl.Main"
f. mvn exec:java -Dexec.mainClass="ch8.order.validating.dsl.Main"

Running the example:
--------------------

From folder "trading", do the following:-


a. mvn clean
b. mvn compile
c. mvn exec:java -Dexec.mainClass="ch8.trading.parsing.dsl.Main"
d. mvn exec:java -Dexec.mainClass="ch8.trading.semantic.dsl.Main"
e. mvn exec:java -Dexec.mainClass="ch8.trading.ssi.dsl.Main"
f. mvn exec:java -Dexec.mainClass="ch8.trading.validating.dsl.Main"
